# Movie-Explorer-Dashboard

## Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/movie-explorer-dashboard.git
   cd movie-explorer-dashboard
2. Install dependencies:
   ```sh
   npm install
2. Run the project::
   ```sh
   npm start
